# Trung Fuzzy - React TypeScript

Project đã được chuyển từ bộ HTML/CSS/JS/IMG Fuzzy sang cấu trúc React + TypeScript + Vite.

## Cách chạy

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```
